## 맞춤형 AI 작사 작곡 서비스
- 사용자의 목소리에 어울리는 장르를 추천해주고 작사 작곡 결과를 보여주는 서비스
## 개발 기간
- 22.04 ~ 09 6개월
  
## 수행 내용
AI분야
- 가사 데이터를 모아 파인튜닝을 통해 KoGPT에게 노래에 들어갈 키워드를 주면 해당 내용으로 이뤄진 가사를 만들 수 있게 파인튜닝
- KoGPT에 장르별 토큰 <트로트>, <발라드> 등 을 추가하여 해당 토큰이 처음 입력값으로 들어오면 해당 장르의 가사를 출력할 수 있게 파인튜닝
